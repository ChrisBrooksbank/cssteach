# Typography & Color

## Overview

Teach font styling, text layout, and color systems through interactive tutorials, reference, and challenges.

## User Stories

- As a learner, I want tutorials on fonts and text styling so I can make text look good
- As a learner, I want a color reference showing different color formats so I can pick the right one
- As a learner, I want challenges where I style text to match a target design

## Requirements

### Tutorials

- [ ] Font properties tutorial (family, size, weight, style)
- [ ] Text layout tutorial (line-height, letter-spacing, text-align, text-transform)
- [ ] Web fonts tutorial (loading and using custom fonts)
- [ ] Color formats tutorial (hex, rgb, hsl, named colors, oklch)
- [ ] CSS custom properties for theming tutorial

### Reference

- [ ] Typography property reference with visual examples
- [ ] Color format comparison chart
- [ ] Font stack recommendations (system fonts, web-safe, popular pairings)

### Challenges

- [ ] Match the typography challenge (recreate a text style)
- [ ] Build a color palette challenge (create harmonious colors using hsl)
- [ ] Theme switcher challenge (light/dark using custom properties)

## Acceptance Criteria

- [ ] Tutorials let users tweak values with sliders/inputs and see live results
- [ ] Color reference shows visual swatches alongside code
- [ ] Challenges show target vs user output side-by-side
- [ ] All content is short, scannable, low cognitive load

## Out of Scope

- Advanced variable fonts (font-variation-settings deep dive)
- Color management / ICC profiles
