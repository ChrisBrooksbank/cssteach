# Animations & Transitions

## Overview

Teach CSS animations, transitions, and transforms through interactive tutorials, reference, and challenges.

## User Stories

- As a learner, I want tutorials on transitions and animations so I can add motion to my pages
- As a learner, I want a reference for timing functions and transform values
- As a learner, I want challenges where I create specific animations

## Requirements

### Tutorials

- [ ] Transitions tutorial (property, duration, timing-function, delay)
- [ ] Transforms tutorial (translate, rotate, scale, skew, 3D basics)
- [ ] Keyframe animations tutorial (@keyframes, animation properties)
- [ ] Combining transforms and animations tutorial
- [ ] Reduced motion and accessibility tutorial (prefers-reduced-motion)

### Reference

- [ ] Transition property reference with interactive demos
- [ ] Transform function reference with visual examples
- [ ] Timing function gallery (ease, linear, cubic-bezier visualizer)
- [ ] Common animation patterns (fade, slide, bounce, spin)

### Challenges

- [ ] Build a hover effect challenge
- [ ] Create a loading spinner challenge
- [ ] Animate a page transition challenge
- [ ] Build an interactive button challenge (with state transitions)

## Acceptance Criteria

- [ ] Tutorials have play/pause/restart controls for animations
- [ ] Reference shows animations running live (not static images)
- [ ] Challenges compare user animation to target (visual + timing)
- [ ] All content is short, scannable, low cognitive load

## Out of Scope

- JavaScript animation libraries (GSAP, Framer Motion)
- Web Animations API
- Scroll-driven animations (could be future spec)
