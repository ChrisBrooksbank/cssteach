# Selectors & Specificity

## Overview

Teach how CSS selectors work, the cascade, and specificity rules through interactive examples and challenges.

## User Stories

- As a learner, I want tutorials explaining selector types so I understand how to target elements
- As a learner, I want a specificity calculator/reference so I can debug styling conflicts
- As a learner, I want challenges that test my selector knowledge

## Requirements

### Tutorials

- [ ] Basic selectors tutorial (element, class, ID, universal)
- [ ] Combinator selectors tutorial (descendant, child, sibling)
- [ ] Pseudo-class selectors tutorial (:hover, :nth-child, :focus, etc.)
- [ ] Pseudo-element selectors tutorial (::before, ::after, ::first-line)
- [ ] Specificity and cascade tutorial (how browsers resolve conflicts)

### Reference

- [ ] Selector syntax quick-reference table
- [ ] Specificity scoring guide with visual breakdown
- [ ] Common selector patterns and when to use them

### Challenges

- [ ] Select the right elements challenge (given HTML, write selectors)
- [ ] Specificity battle challenge (predict which style wins)
- [ ] Debug a broken stylesheet challenge (fix selector issues)

## Acceptance Criteria

- [ ] Tutorials highlight matched elements in real-time as user types selectors
- [ ] Specificity reference shows scoring visually (e.g., color-coded columns)
- [ ] Challenges give instant feedback on correctness
- [ ] All content is short, scannable, low cognitive load

## Out of Scope

- CSS-in-JS or styled-components
- Preprocessor selectors (Sass nesting syntax as primary teaching)
