# Layout (Flexbox & Grid)

## Overview

Teach CSS layout techniques through interactive tutorials, a reference guide, and hands-on challenges.

## User Stories

- As a learner, I want step-by-step tutorials on flexbox and grid so I can understand how layout works
- As a learner, I want a quick-reference for layout properties so I can look things up while coding
- As a learner, I want challenges where I recreate layouts by writing CSS so I can practice

## Requirements

### Tutorials

- [ ] Flexbox basics tutorial (direction, justify, align)
- [ ] Flexbox wrapping and grow/shrink tutorial
- [ ] CSS Grid basics tutorial (columns, rows, template areas)
- [ ] Grid placement and spanning tutorial
- [ ] Responsive layout tutorial (combining flex/grid with media queries)

### Reference

- [ ] Flexbox property reference with visual examples
- [ ] Grid property reference with visual examples
- [ ] Common layout patterns (holy grail, sidebar, card grid)

### Challenges

- [ ] Center an element challenge (multiple approaches)
- [ ] Build a navigation bar challenge
- [ ] Build a card grid challenge
- [ ] Build a holy grail layout challenge

## Acceptance Criteria

- [ ] Each tutorial has a live CSS editor with instant preview
- [ ] Reference entries show property values with visual demos
- [ ] Challenges compare user output to a target image/layout
- [ ] All content is short, scannable, low cognitive load

## Out of Scope

- Float-based layouts
- Table-based layouts
- CSS frameworks (Bootstrap, Tailwind utility classes as teaching content)
